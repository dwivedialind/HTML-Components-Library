Color Values

HTML RGB and RGBA colors

rgb(red,green,blue),each parameter inside defines the intensity of the color with a value between 0 and 255. This means 256*256*256 possible colors.
for black: rgb(0,0,0)/for white: rgb(255,255,255)

Shades of gray are often defined using equal values for all the three parameters

rgba have special parameter i.e, alpha which specified opacity of the Color
Range of alpha; 0.0(transparent) to 1.0(not transparent at all)

HEX Color values

#rrggbb, where rr(red),gg(green),bb(blue) are hexadecimal values between 00 and ff being the highest and 00 lowest.
for black: #000000 and for white: #ffffff.
Shades of gray are obtained using equal value for all three parameters.

HSL and HSLA color Values

hsl stands for hue,saturation and lightness
Hue is a degree on the color wheel from 0 t 360. 0 is red and 120 is green and 240 is blue.
Saturation is a percentage value, 0% means shade of gray and 100% is the full color.
Lightness is also a percentage value which means how much light you want to give the color,0% is black, and 100% is white.

Shades of gray are defined by setting the hue and saturation to 0, and adjusting the lightness from 0% to 100%

HTML Links- The target Attribute

target attribute specifies where to open the linked document.

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab
  • _parent - Opens the document in the parent frame
  • _top - Opens the document in the full body of the window
